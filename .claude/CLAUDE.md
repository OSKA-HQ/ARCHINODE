## 1. ARCHINODE란?

**한 줄 정의:** 글로벌 건축·디자인 브랜드를 한국 시장과 연결하는 B2B/B2C 플랫폼

**벤치마크:** Archiproducts.com (이탈리아, 305K+ 제품, 3,500+ 브랜드, 3.7M 유저)

**도메인:** archinodekr.com

**GitHub:** https://github.com/OSKA-HQ/ARCHINODE

**운영 주체:** 비비들리바이브 (VIVIDELIVIBE) / 대표: 박승리 / 사업자등록번호: 640-03-02879

## 2. 플랫폼의 핵심 가치

ARCHINODE는 **세 가지 유저 타입**을 연결하는 플랫폼이다:

### 해외 브랜드 (Brand)
- 한국 시장에 진출하고 싶은 글로벌 건축·디자인 브랜드
- 국내 딜러와 설계사들이 자사 제품을 알고, 설계에 반영하거나, 딜러십을 맡아주기를 원함
- 본인의 브랜드 페이지를 직접 꾸미고, 제품 정보를 직접 등록하고, 매거진(홍보 기사)을 직접 작성
- 국내 시장의 반응(좋아요, 문의)을 실시간으로 확인

### 국내 전문가 (Professional)
- 설계사, 인테리어 업체, 디자이너, 딜러 등 한국 건축·디자인 업계 종사자
- 해외 브랜드를 손쉽게 탐색하고, 영감을 얻고, 필요한 제품을 찾는 플랫폼
- 마음에 드는 제품/브랜드에 좋아요를 눌러 관심 표현
- 브랜드와 직접 소통할 수 있는 채팅/프로젝트방을 통해 이메일 없이 문의 가능

### ARCHINODE 어드민 (Admin)
- 브랜드 입점 신청 승인/반려
- 브랜드가 올린 콘텐츠(브랜드 페이지, 제품 정보, 매거진) 검수 후 배포
- 플랫폼 전체 운영 및 품질 관리

## 3. 핵심 사용자 흐름

### 브랜드 입점 흐름
1. 해외 브랜드가 입점 신청
2. ARCHINODE 어드민이 승인
3. 브랜드 담당자에게 계정 발급 (또는 승인 후 활성화)
4. 브랜드 담당자가 브랜드 관리 대시보드에서:
   - 브랜드 페이지 편집 (로고, 배너, 소개, 연혁 등)
   - 제품 등록 (이미지, 스펙, 카테고리, 설명)
   - 매거진/뉴스 기사 작성 (자사 홍보용)
5. 어드민이 등록된 콘텐츠 검수 → 승인 시 사이트에 공개

### 국내 전문가 흐름
1. 회원가입 (업종, 회사명 등 프로필)
2. 브랜드/제품 탐색 (카테고리, 검색, 필터)
3. 마음에 드는 제품/브랜드에 좋아요
4. 궁금한 점이 있으면 채팅/프로젝트방에서 브랜드에 직접 문의
5. 자료 다운로드 (카탈로그 PDF, 향후 BIM/3D)

### 어드민 흐름
1. 입점 신청 관리 (승인/반려)
2. 콘텐츠 검수 큐 (대기 중인 제품, 매거진, 브랜드 페이지 변경)
3. 사용자 관리 (브랜드 계정, 전문가 계정)
4. 사이트 통계 및 모니터링

### 한울님 추가 아이디어 (2026-04-11)

**① 한국 딜러 연결 기능 (우선순위: 높음 — Phase 2A 보강)**
- 브랜드 페이지에 한국 공식 딜러 안내 섹션 추가
- 예시: Kettal 브랜드 페이지 → "한국 공식 딜러: Nexus" 안내
- Firestore `brands` 컬렉션에 `koreanDealer` 필드 추가 (딜러명, 연락처, 웹사이트, 주소)
- 브랜드 대시보드에서 딜러 정보 입력 가능하게
- 한국 디자이너가 브랜드 제품을 설계에 반영하고 싶을 때 → 딜러를 바로 찾을 수 있게

**② 소셜 인터랙션 — 좋아요 + 댓글 (우선순위: 좋아요 Phase 2B / 댓글 Phase 3)**
- 인스타그램처럼 제품에 좋아요 + 댓글 기능
- 좋아요: Phase 2B에서 구현 (스키마 준비됨)
- 댓글: Phase 3에서 채팅/프로젝트방과 함께 도입 (모더레이션 필요)
- 브랜드가 신제품 올리면 → 국내 디자이너/건축가들이 반응할 수 있는 구조

**③ 디자이너 페이지 (우선순위: Phase 4-5)**
- 브랜드와 콜라보하는 디자이너 전용 페이지
- 예시: 디자이너 "노만" → Kettal, Ethimo 등 콜라보 브랜드 목록 + 콜라보 제품
- Firestore `designers` 컬렉션 신설, 제품에 `designerId` 연결
- 한국 가구회사도 디자이너와 콜라보 유도 가능 → 플랫폼의 가치 확장
- 브랜드 20~50개 이상 쌓인 후 도입이 현실적

**④ 카탈로그/도면 무료 다운로드 (우선순위: Phase 3)**
- 아키프로덕트처럼 PDF 카탈로그, 기술 도면 무료 다운로드
- 단, 회원가입(로그인) 후에만 다운로드 가능 → 회원가입 유도 + 브랜드에 관심 데이터 제공
- Firebase Storage 활용 (스키마에 `catalogPdf` 필드 이미 준비됨)
- 아키프로덕트는 Firebase 기반 아님 (전통적 서버 기반 아키텍처, 2000년대 이탈리아 기업)
